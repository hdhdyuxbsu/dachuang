🔗 KEY CONCEPTS
================================================================================

Bitmap Format:
   • 12×12 pixels per character
   • 12 rows × 1 uint16_t per row
   • Bits [15:4] = pixel data
   • Bits [3:0] = padding (always 0)
   • MSB first = leftmost pixel

Pixel Extraction:
   pixel = (row_data >> (15 - column)) & 1;

Lookup:
   const uint16_t *bitmap = get_char_bitmap(code_point);

Iteration:
   for (int i = 0; char_bitmaps[i].code_point != 0; i++) { ... }

Display:
   For each filled pixel, call your LCD's drawPixel() function

❓ FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Which file do I use in my ESP32 code?
A: char_bitmap.h - Include it and use get_char_bitmap()

Q: Do I need to run the scripts?
A: No, unless you want to change fonts or add characters

Q: What does 0x3E00 mean?
A: It's the 12-pixel data for one row (see VISUAL_PREVIEW.txt)

Q: How do I extract a pixel?
A: pixel = (bitmap[row] >> (15 - col)) & 1;

Q: Can I change the character size?
A: Yes, edit generate_bitmap.py and change BITMAP_SIZE

Q: Can I add more characters?
A: Yes, add to CHARACTERS list in generate_bitmap.py

Q: What about non-Chinese characters?
A: The system works with any Unicode character that your font supports

Q: How much memory does this use?
A: ~700 bytes runtime + ~10 KB in flash (very small!)

🔍 TROUBLESHOOTING GUIDE
================================================================================

Problem: Characters look wrong
Solution: See VISUAL_PREVIEW.txt to verify bitmap data
          Try different font in generate_bitmap.py

Problem: Memory issues on ESP32
Solution: This uses only ~700 bytes, shouldn't be a problem
          Check your other code allocations

Problem: Script won't run
Solution: Install Python 3
          Run: pip install Pillow
          Check font paths in generate_bitmap.py

Problem: Font not found
Solution: Copy fonts from C:\Windows\Fonts\ to project
          Or update font paths in generate_bitmap.py

Problem: Need different character set
Solution: Edit CHARACTERS list in generate_bitmap.py
          Add your Unicode code points
          Run script to regenerate
